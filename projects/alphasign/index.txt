{{{alphasign}}} is a Python library for [[http://www.alpha-american.com|Alpha American]] LED signs, such as the [[http://www.betabrite.com|Betabrite]]. It
implements the
[[http://www.alpha-american.com/p-alpha-communications-protocol.html|Alpha Communications Protocol]].

The picture on the right shows a Betabrite Prism displaying the time and
temperature updated by a program using the alphasign library.

== Installation ==

{{{alphasign}}} is [[http://pypi.python.org/pypi/alphasign|listed]] in the
[[http://pypi.python.org|Python Package Index]]. It can be installed with
[[http://peak.telecommunity.com/DevCenter/setuptools|easy_install]]:

{{{
sudo easy_install -U alphasign
}}}

== Documentation ==

The documentation is [[/projects/alphasign/docs|available online]]. This project is documented with [[http://sphinx.pocoo.org/|Sphinx]].
